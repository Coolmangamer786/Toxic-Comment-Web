## Run Locally

Clone the project

```bash
  git clone https://github.com/Snehasis4321/Toxic-Comment-Web.git
```

Go to the project directory

```bash
  cd Toxic-Comment-Web
```

Install dependencies

```python
  pip install -r requirements.txt
```

Start the server

```python
  python main.py
```

For Production 
```python
  gunicorn main:app
```
